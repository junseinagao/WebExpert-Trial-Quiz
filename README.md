# WebExpert コース体験会のクイズアプリ

## このレポジトリの使い方

- master branch 完成例
- black branch 穴抜き版

## 使う技術

- HTML
- CSS
- JavaScript

## 分担例

1. `index.html`で`<h1>`, `<img>`, `<button>`. `<div>`タグとその textContent を書く (id 属性を付ける)
2. `main.js`で`const ○○ document.getElementById("ID名")`で要素を取得する。
3. `○○.onclick = function () {}`でボタンを押した時の処理を書く。
